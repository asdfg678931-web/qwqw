비정형 환경에서도 안정적으로 자세를 유지하는 실시간 밸런싱 제어 알고리즘과 라이다(LiDAR) 기반 3D 지형 매핑 기술이 집약된 플래그십 모델입니다. 고출력 모터와 자체 설계된 관절 구조를 통해 극한의 재난 현장에서 인명 구조 임무를 수행합니다.

실시간 밸런싱 제어 LiDAR 3D 매핑 C/C++

2. 지능형 F&B 맞춤형 서빙 및 관제 로봇

현장 경험에서 탄생한 자율주행과 AI 비전 기반 스마트 관제

아웃백 스테이크하우스처럼 바쁘고 복잡한 외식업 현장에서 직접 근무하며 겪었던 동선 꼬임과 돌발 상황들을 해결하기 위해 기획되었습니다. 단순한 경로 이동을 넘어 매장 내 장애물을 유연하게 회피하며, 실제 매장 경험을 바탕으로 기획된 꼭 필요한 기능만 압축한 웹 대시보드(Firebase 연동)를 통해 관리자가 매장 전체를 직관적으로 관제할 수 있습니다.

3. 스마트팩토리 통합 AI 비전 솔루션

초저지연 통신 기반의 실시간 딥러닝 불량 검출

공장 조립 라인을 지나가는 부품의 미세한 불량을 즉각적으로 찾아내고 하드웨어를 제어하는 통합 솔루션입니다. Python/C# 기반으로 구축된 AI 딥러닝 뇌(Brain)와 PLC 등 공장 제어 기기 간의 초저지연 통신을 통해 수율을 극대화합니다.

딥러닝 이미지 분류 PLC 통신 Python / C#

4. 커스텀 가능 데스크탑 AI 반려 로봇

나만의 비서이자 친구, B2C 타겟형 소형 로봇

책상 위에 두고 업무 일정을 관리하거나 감정을 표현하며 상호작용하는 스마트 로봇입니다. 사용자가 직접 코드를 수정하여 로봇의 성격이나 LED 동작을 커스텀할 수 있는 개방성을 지니고 있어, 일반 소비자들의 코딩 교육 및 일상 보조 역할로 탁월한 기획력을 보여줍니다.

음성 인식/합성 아두이노(Arduino) 오픈소스 커스텀

5. 험지 개척용 사족보행 로봇 및 드론 격납 스테이션

플래그십 이족보행 로봇과 연계되는 공중/지상 복합 정찰 플랫폼

이족보행 로봇이 진입하기 전, 붕괴 위험 지역을 먼저 탐색하는 사족보행(Quadruped) 로봇 플랫폼입니다.

C++ 기반의 역운동학(IK) 알고리즘으로 불규칙한 지형에서도 자세를 안정적으로 유지하며, 상단의 무인 드론 격납 스테이션을 통해 공중 수색 드론과 실시간 3D 매핑 데이터를 동기화합니다. 재난구조 휴머노이드의 작전 효율을 극대화하는 선행 정찰 솔루션입니다.

역운동학(IK) 제어 드론 군집 매핑 C++ / ROS2

6. 협동 로봇 암(Arm) 연동 스마트 패키징 솔루션

24시간 무인 가동을 위한 비전 AI 및 모션 제어 통합

Python/C# 딥러닝 비전으로 불량을 검출한 후, 6축 협동 로봇 암과 PLC 통신을 통해 정상품과 불량품을 자동 분류·패키징하는 통합 자동화 시스템입니다.

초당 5개 이상의 물체를 실시간 추적(Object Tracking)하여 생산 라인 중단 없이 24시간 완전 무인 가동이 가능합니다. 스마트팩토리 고도화의 핵심 솔루션입니다.

Object Tracking 로봇 암 모션 제어 공장 자동화 통합
